# bingi-server

A minimal Node.js project scaffolded with TypeScript.

Available scripts:

- `npm run dev` — start in development (ts-node-dev)
- `npm run build` — compile TypeScript to `dist/`
- `npm run start` — run compiled code from `dist/`
- `npm run lint` — run ESLint
- `npm run format` — run Prettier

To run locally:

1. npm install
2. npm run dev
